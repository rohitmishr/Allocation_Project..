PROJECT ALLOCATION TRACKER



The Project Allocation Tracker (PAT) is a single unified platform for providing visibility of resource availability across tracks and utilizing the same information for actionizing the allocations.

The Project Allocation Tracker includes resource and project definition and maintenance, allocation, availability reporting and leadership visibility dashboards.
